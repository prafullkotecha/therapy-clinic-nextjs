# Incident Response Template

## 1. Detection and Triage
- Identify affected tenant(s), user(s), and systems
- Capture timeline and initial indicators
- Start audit log integrity verification for impacted tenant records

## 2. Containment
- Disable affected accounts / rotate credentials
- Isolate impacted components
- Enable heightened monitoring and lockout controls

## 3. Eradication and Recovery
- Remove root cause
- Validate restoration with integrity checks and security tests
- Document remediations and preventive controls

## 4. Notification Workflow
- Internal security escalation
- Compliance/legal review
- HIPAA breach notification process per organizational policy and timelines

## 5. Post-Incident Review
- Root cause analysis
- Control gap tracking
- Action items with owners and due dates
